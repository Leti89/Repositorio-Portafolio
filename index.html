Junio 2022 - Presente

En TechSolutions, lideré el desarrollo de un sistema de gestión de inventarios para una empresa minorista. Utilicé tecnologías como HTML, CSS, JavaScript, y React para el frontend, y Node.js con Express para el backend. Este sistema permitió a la empresa optimizar sus operaciones y mejorar la precisión del inventario. Algunas de mis responsabilidades incluyeron:

  • Desarrollar y mantener componentes de frontend utilizando React.
  • Implementar APIs RESTful con Node.js y Express.
  • Colaborar con el equipo de diseño para asegurar una experiencia de usuario intuitiva.
  • Realizar pruebas unitarias y de integración para asegurar la calidad del código.
  • Participar en reuniones diarias de scrum y en la planificación de sprints.

Gracias a este proyecto, la empresa pudo reducir sus errores de inventario en un 20% y mejorar la eficiencia operativa en un 15%. Esta experiencia me permitió fortalecer mis habilidades técnicas y de trabajo en equipo.
